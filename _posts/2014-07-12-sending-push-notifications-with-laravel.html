With this handy package you can send iOS and Android push notifications from your Laravel app: davibennun/laravel-push-notification.

Following the Installation and Configuration instructions get you up and running but bear this in mind:

NO {% highlight php linenos %} PushNotification::app('your.appname') ->to('your device token goes in this method call') ->send('Hello World'); {% endhighlight %} YES {% highlight php linenos %} PushNotification::app('your.appname') ->to('yourdevicetokengoesinthismethodcall') ->send('Hello World'); {% endhighlight %} Otherwise you'll get something like this: {% highlight php %} Adapter Apns does not supports tokens device {% endhighlight %}
