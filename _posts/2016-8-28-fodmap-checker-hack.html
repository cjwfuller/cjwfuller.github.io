This morning I hacked together a web app in about 2 hours (as you can probably tell). The application just checks whether a food is high FODMAP and if it is, the web app alerts the user.

It's hosted on S3 here (it doesn't even have a domain yet. Update: Now has a domain: www.fodmapchecker.com

Technology is nothing fancy at the moment:
  • Python SimpleHTTPServer for local development
  • Gulp task runner
  • jQuery
If anyone actually visits the site then I'll probably introduce:
  • React
  • Elasticsearch with typeahead
