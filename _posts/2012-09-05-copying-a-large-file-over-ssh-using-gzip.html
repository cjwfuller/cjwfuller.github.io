The following will copy a file from a remote server to your local machine:

ssh some-user@some-host gzip -c /path/to/remote/file/to/compress compressed_file.gz

On my server, I have a user 'some-user' that is a sudoer. Some files on my server are only accessible by root so if I want to get these files to my local machine, I need to send the sudo command to the remote server:

ssh some-user@some-host "echo 'gzip -c /path/to/remote/file/to/compress' | sudo sh" > compressed_file.gz
