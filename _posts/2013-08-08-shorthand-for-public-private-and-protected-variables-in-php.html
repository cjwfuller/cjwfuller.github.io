I realised today that you can do this:
[code lang="php"]
class Monster {
private
$colour;
public
$size,
$isEvil;
}
[/code]
instead of this:

[code lang="php"]
class Monster {
private $colour;
public $size,
public $isEvil;
}
[/code]
in PHP which cool. It reminds me of C++. I guess it can come in handy if you have many variables that are all private or all protected, ...
